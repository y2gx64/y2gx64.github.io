## What is a stack
A stack is like a deck of cards, it is a last in first out data structure. Think of it as a temporary storage for local variables, return addresses, etc.  To add something onto the stack:

```
PUSH [source_register]
```
When pushing, this will increase the ESP register which tracks the top of the stack. 


To remove something:
```
POP [destination_register]
```
When popping, this will decrease the ESP register which tracks the top the stack.

### Example
```
section .data
  msg db "hello"

section .text
  global _start

_start:
mov rax, 1
push rax
pop rax
call END

END:
mov rax, 60
syscall 
ret
```
