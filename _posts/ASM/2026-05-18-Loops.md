## How to make a loop
A loop consists of a counter, a conditional jump and the code that is supposed to run per loop. 

### Example
```
section .data
  text db `loop\n`
  textlen equ $ - text

section .text 
  global _start

_start:
  mov cl, 10
  OUTPUT: 
  mov rdi, 1
  mov rsi, text
  mov rdx, textlen
  mov rax, 1
  push rcx
  syscall 
  pop rcx
  dec cl
  jnz OUTPUT
  
  mov rax, 60
  syscall

```
The counter is `rcx`, as i want the code to loop ten times, i moved `10` into `rcx`.
